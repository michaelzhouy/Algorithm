# 打印从1到最大的n位数

## 说明
输入数字 n，按顺序打印出从 1 到最大的 n 位十进制数。比如输入 3，则打印出 1、2、3 一直到最大的 3 位数 999。

## 示例
```
输入: n = 1
输出: [1,2,3,4,5,6,7,8,9]
```

## 解1
```python
class Solution:
    def printNumbers(self, n: int) -> List[int]:
        res = 0
        for i in range(n):
            res += 9 * 10 ** i
        return list(range(1, res + 1))
```

## 解2
```python
class Solution:
    def printNumbers(self, n: int) -> List[int]:
        return list(range(1, 10 ** n))
```
