# 56-2-数组中数字出现的次数

## 说明
在一个数组 $nums$ 中除一个数字只出现一次之外, 其他数字都出现了三次. 请找出那个只出现一次的数字.

## 示例
```
输入：nums = [3,4,3,3]
输出：4

输入：nums = [9,1,7,9,7,9,7]
输出：1
```

## 解1 哈希

### 思路
利用字典

### 复杂度
1. 时间复杂度$O(N)$
2. 空间复杂度$O(N)$

```python
class Solution:
    def singleNumber(self, nums: List[int]) -> int:
        d = {}
        for num in nums:
            if d.get(num) is not None:
                d[num] += 1
            else:
                d[num] = 1
        for k, v in d.items():
            if v == 1:
                return k
```
