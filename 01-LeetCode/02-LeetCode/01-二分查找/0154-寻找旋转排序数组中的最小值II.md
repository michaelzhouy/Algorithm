# 0154-寻找旋转排序数组中的最小值II

## 说明
假设按照升序排序的数组在预先未知的某个点上进行了旋转。

( 例如，数组 [0,1,2,4,5,6,7] 可能变为 [4,5,6,7,0,1,2] )。

请找出其中最小的元素。

注意数组中可能存在重复的元素

## 示例
```
输入: [1,3,5]
输出: 1

输入: [2,2,2,0,1]
输出: 0
```

## 解1 二分查找

### 思路
1. 最小值出现在右边递增排序的数组中
2. 每个中间值与右边的值比较

### 复杂度
1. 时间复杂度$O(logN)$
2. 空间复杂度$O(1)$

```python
class Solution:
    def findMin(self, nums: List[int]) -> int:
        left, right = 0, len(nums) - 1
        while left < right:
            mid = (left + right) >> 1
            if nums[mid] > nums[right]:
                left = mid + 1
            elif nums[mid] < nums[right]:
                right = mid
            elif nums[mid] == nums[right]:
                right -= 1
        return nums[left]
```
