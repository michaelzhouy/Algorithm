# 0144-二叉树的前序遍历

## 说明
给你二叉树的根节点 root ，返回它节点值的 前序 遍历。

## 示例
```
输入：root = [1,null,2,3]
输出：[1,2,3]

输入：root = []
输出：[]

输入：root = [1]
输出：[1]

输入：root = [1,2]
输出：[1,2]

输入：root = [1,null,2]
输出：[1,2]
```

### 复杂度
1. 时间复杂度O(N)
2. 空间复杂度O(N)

```python
class Solution:
    def preorderTraversal(self, root: TreeNode) -> List[int]:
        if not root:
            return []
        def preOrder(root):
            if not root:
                return
            res.append(root.val)
            preOrder(root.left)
            preOrder(root.right)
        
        res = []
        preOrder(root)
        return res
```
