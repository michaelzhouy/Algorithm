# 0137-只出现一次的数字II

## 说明
给定一个非空整数数组，除了某个元素只出现一次以外，其余每个元素均出现了三次。找出那个只出现了一次的元素

## 示例
```
输入: [2,2,3,2]
输出: 3

输入: [0,1,0,1,0,1,99]
输出: 99
```

## 解 哈希表

### 复杂度
1. 时间复杂度$O(N)$
2. 空间复杂度$O(N)$

```python
class Solution:
    def singleNumber(self, nums: List[int]) -> int:
        dic = {}
        for num in nums:
            if num not in dic:
                dic[num] = 1
            else:
                dic[num] += 1
        for k, v in dic.items():
            if v == 1:
                return k
```
