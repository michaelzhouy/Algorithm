# 0371-两整数之和

## 说明
不使用运算符 `+` 和 `-` ​​​​​​​，计算两整数 ​​​​​​​`a` 、`b` ​​​​​​​之和

## 示例
```
输入: a = 1, b = 2
输出: 3

输入: a = -2, b = 3
输出: 1
```

## 解

### 思路
```
4+5

与:
 100
 101
-----
 100

异或
 100
 101
-----
 001

(100 << 1) + 001
```

### 复杂度
1. 时间复杂度$O(1)$
2. 空间复杂度$O(1)$

```python
class Solution:
    def getSum(self, a: int, b: int) -> int:
        return ((a + b) << 1) + (a ^ b)
```
