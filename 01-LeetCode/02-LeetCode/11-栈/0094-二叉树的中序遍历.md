# 0094-二叉树的中序遍历

## 说明
给定一个二叉树的根节点 root ，返回它的 中序 遍历。

## 示例
```
输入：root = [1,null,2,3]
输出：[1,3,2]
```

## 解1 递归

```python
class Solution:
    def inorderTraversal(self, root: TreeNode) -> List[int]:
        def dfs(root):
            if not root:
                return
            dfs(root.left)
            res.append(root.val)
            dfs(root.right)
        
        res = []
        dfs(root)
        return res
```

## 解2 循环

```python
class Solution:
    def inorderTraversal(self, root: TreeNode) -> List[int]:
        if not root:
            return []
        cur, stack, res = root, [], []
        while stack or cur:
            while cur:
                stack.append(cur)
                cur = cur.left
            tmp = stack.pop()
            res.append(tmp.val)
            cur = tmp.right
        return res
```
