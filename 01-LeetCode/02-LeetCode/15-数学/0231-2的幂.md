# 0231-2的幂

## 说明
给定一个整数，编写一个函数来判断它是否是 2 的幂次方

## 示例
```
输入: 1
输出: true
解释: 20 = 1

输入: 16
输出: true
解释: 24 = 16

输入: 218
输出: false
```

## 解

```python
class Solution:
    def isPowerOfTwo(self, n: int) -> bool:
        if n < 1:
            return False
        if n == 1:
            return True
        if n & 1 == 1: # 奇数
            return False
        while n % 2 == 0:
            n = n >> 1 # 除以2
        return n == 1
```
