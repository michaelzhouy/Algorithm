# 0504-七进制数

## 说明
给定一个整数，将其转化为7进制，并以字符串形式输出。

## 示例
```
输入: 100
输出: "202"

输入: -7
输出: "-10"
```

## 解

```python
class Solution:
    def convertToBase7(self, num: int) -> str:
        if num == 0:
            return '0'
        flag = 1
        if num < 0:
            flag = -1
            num = -num
        res = ''
        while num:
            num, remainder = divmod(num, 7)
            res += str(remainder)
        return res[::-1] if flag == 1 else '-' + res[::-1]
```
