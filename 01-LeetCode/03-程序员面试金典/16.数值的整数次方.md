# 16.数值的整数次方

## 解

```python
ass Solution:
    def myPow(self, x: float, n: int) -> float:
        # n<0，x=1/x，n=-n
        if n < 0:
            n = -n
            x = 1 / x
        # n==0，返回1
        if n == 0:
            return 1
        # n为奇数，x * (x * x) ^ (n // 2)
        elif n % 2 == 1:
            return x * self.myPow(x * x, n // 2)
        # n为偶数，(x * x) ^ (n // 2)
        else:
            return self.myPow(x * x, n // 2)
```
